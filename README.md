# Vedant Rakhonde Portfolio

Live site: https://ivedx1.github.io

## Local Development

1. Open the project in VS Code.
2. Use Live Server (or any static server) to preview `index.html`.
3. Validate desktop and mobile layouts before publishing.

## Pre-Release Checklist

1. Content review:
	Update project dates, links, and section copy.
2. Accessibility review:
	Verify keyboard navigation, readable contrast, and reduced-motion behavior.
3. Performance review:
	Confirm animations are smooth and no console errors are present.
4. Cross-browser review:
	Test in Chrome, Safari, and a mobile browser.
5. SEO/share metadata:
	Validate title, description, and Open Graph tags.

## Deployment

1. Commit and push to the `main` branch.
2. Confirm GitHub Pages build succeeds.
3. Smoke-test the deployed URL:
	https://ivedx1.github.io

## Rollback

1. Revert the problematic commit locally.
2. Push the revert commit to `main`.
3. Re-check the deployed site.
